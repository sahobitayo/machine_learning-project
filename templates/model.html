Neural Network Model Predicting Median Magnitude

Is the predicated earthquake magnitude greater than or equal to the median, 5.4?
(1 for yes and 0 for no)

1. Import the data
2. Split the data into features and labels, (X and y)
3. Split the data into test, training, validation (70%, 15%, 15%)
4. Did not scale the features due to latitudes and longitudes (creating vectors using lat and long subdivided by the hemispheres, prime meridian and equator) – hard to conceptualize lat and long as independent variables. Dropped decimals and rounded to nearest tens.
5. Sequential model in Keras
6. Define model - dense layers (neurons and activation - Rectified Linear Unit)
7. Compile and Fit model



• Input Layer: 5 features (month, latitude, longitude, depth, magnitude)
• Hidden layer 1: 32 neurons, ReLU activation
• Hidden layer 2: 32 neurons, ReLU activation
• Output Layer: 1 neuron, Sigmoid activation (squashing function limit the output to a range between 0 and 1)
• Compile - Optimizer = sgd (stochastic gradient descent - small subset), loss = binary crossentropy (loss function for output values 1 or 0)
• Fit - Batch size 32, Epochs 100
• Batch size 32, Epochs 100
• Loss = 0.5678161978721619, accuracy = 0.7360464334487915
